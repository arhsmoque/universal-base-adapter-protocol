# frontend-design-gold

Import-ready Claude Skill package for context-adaptive frontend design direction, review, redesign, and implementation.

## Claude import

1. Zip structure should be:
   ```text
   frontend-design-gold.zip
   └── frontend-design-gold/
       ├── SKILL.md
       └── resources/
           ├── frontend-evaluation-rubric.md
           └── reference-anchors.md
   ```
2. In Claude, go to Customize > Skills.
3. Choose Create skill / Upload a skill.
4. Upload `frontend-design-gold.zip`.
5. Enable the skill.

## Suggested trigger prompts

- "Use frontend-design-gold to review this UI."
- "Use frontend-design-gold to create design direction for this frontend."
- "Use frontend-design-gold to redesign this dashboard."
- "Use frontend-design-gold to implement this component accessibly."
