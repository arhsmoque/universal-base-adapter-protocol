Universal Base/Adapter Protocol Validation Site

A reviewer-facing map of the v1.6 baseline: what it governs, why it exists, and how to check whether a component follows the protocol without reading every file first.

Core Thesis

The user expresses intent. The core owns invariant behavior. Adapters translate runtime details. Rules hold tunable policy. Metadata records provenance. Results describe observable reality.
